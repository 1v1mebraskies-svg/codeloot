================================================================================
                            TECHNICAL DETAILS
================================================================================

SEARCH ALGORITHM:
- For each game card, combines: name + title + description + metadata + alt text + link text
- Converts to lowercase for case-insensitive matching
- Uses String.includes() for partial matching
- O(n) complexity where n = number of cards
- With 26 cards: <1ms search time
- Scales well to 50+ games

GAME DETECTION:
- Queries DOM for elements with [data-game-card] attribute
- Runs on page load AND after each filter
- Works for dynamically added cards (if added via DOM)

SEARCHABLE CONTENT PER CARD:
1. data-game-name attribute (explicit game name)
2. <h3> text (game title)
3. <p> text (game description)
4. .card-meta text (code counts, reward types)
5. img alt text (additional context)
6. .text-link text ("View codes" / "View status")

EVENT HANDLERS:
- Input event: Live filtering as user types
- Submit event: Navigate to first matching game
- Keydown event: Escape to clear search

DATALIST AUTO-POPULATION:
- Gets all unique game names from cards
- Updates datalist <option> elements
- Provides autocomplete suggestions
- Updates on page load

================================================================================
                         HOW TO ADD NEW GAMES
================================================================================

NO CODE CHANGES NEEDED! Simply:

1. Add game card to index.html with:
   - class="game-card card"
   - data-game-card attribute
   - data-game-name="Game Name" (unique)
   
2. Include searchable content:
   - <h3>Game Name</h3>
   - <p>Game description with keywords</p>
   - <div class="card-meta">...</div>
   - <img alt="Game description">

3. That's it! The search system will:
   - Auto-detect the new card
   - Include it in all searches
   - Add it to datalist suggestions
   - NO search.js modifications needed
   - Future-proof and scalable

Example:
<a class="game-card card" href="games/my-game.html" data-game-card data-game-name="My Game">
    <img src="assets/img/my-game.png" alt="My Game description">
    <div class="game-card-body">
        <div class="card-meta"><span>10 active</span></div>
        <h3>My Game</h3>
        <p>Game description with keywords</p>
        <span class="text-link">View codes</span>
    </div>
</a>
