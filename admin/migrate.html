Migrate Existing Games

Use this tool to manually add your existing games to the JSON database. This is a one-time setup process.

Instructions

  1. Click "Generate JSON Template" to create a base structure
  2. Edit the JSON in the textarea to add your existing games
  3. Each game needs: name, slug, description, category, codes array
  4. Click "Download JSON File" to save it
  5. Move the downloaded file to data/games.json
  6. Open admin/index.html to use the dashboard
